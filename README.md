## 👨‍💻 About Me

I am a backend engineer working with the Amar Ujala technology team, focused on designing and deploying **automation systems for high-volume operational workflows**.

I have architected and implemented a **production-grade RO automation pipeline** that eliminates manual intervention and ensures **reliable, high-speed document processing**.

My current focus is on building a **scalable OCR-driven document intelligence system**, capable of extracting structured data across **2000+ document formats**, with continuous improvements in:

- Layout-aware extraction  
- Preprocessing optimization  
- Schema validation  
- Accuracy and robustness

## ⚙️ Production System: RO Automation Pipeline

A fully automated backend system for processing release order (RO) documents at scale.

### 🚀 Core Capabilities

- Zero manual intervention workflow  
- Multi-format document handling (2000+ layouts)  
- Dynamic parser routing and classification  
- OCR + rule-based + schema-driven extraction  
- Database validation and deduplication  
- Automated failure handling with notification system
